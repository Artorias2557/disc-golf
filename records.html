To share rounds between devices, you need a free Firebase Realtime Database.
Then paste the URL into both index.html and records.html where it says YOUR_FIREBASE_DB_URL_HERE.
  1. Go to firebase.google.com and sign in with your Google account.
  2. Click Add project, name it anything (e.g. "disc-golf"), skip analytics.
  3. In the left sidebar, click Build → Realtime Database.
  4. Click Create Database, choose a region, then select Start in test mode.
  5. Copy the URL shown (looks like https://disc-golf-default-rtdb.firebaseio.com).
  6. Paste it into both HTML files replacing YOUR_FIREBASE_DB_URL_HERE, then re-upload to GitHub.
